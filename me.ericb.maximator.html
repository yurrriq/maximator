make-locator

(make-locator f & {:keys [locales file-mode], :or {locales [:en]}})

Returns a locator from either a String file path, a File object or an InputStream. This locator is threadsafe. Takes two optional arguments:

{:locales   [:en]              ; List of recognised locales for default :name.
 :file-mode (or :memory-mapped ; default
                :memory)}
